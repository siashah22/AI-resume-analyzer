# AI Resume Analyzer

An AI-powered web application that analyzes resumes and provides intelligent feedback using Natural Language Processing and Generative AI.

---

## Features

- Upload PDF resume
- Extract and clean text from resumes
- Skill extraction and matching with job descriptions
- AI-generated feedback (Strengths, Weaknesses, Suggestions)
- Interactive web interface using FastAPI + HTML

---

## Tech Stack

- Python
- FastAPI
- PyPDF2
- NLP (text processing)
- LLaMA 3 (Ollama)
- HTML, JavaScript

---

## How It Works

1. Upload resume (PDF)
2. Extract text from PDF
3. Clean and preprocess text
4. Extract skills using NLP
5. Match with job description skills
6. Generate AI-based feedback using LLM
